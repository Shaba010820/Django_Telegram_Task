1. Описание проекта

Telegram ToDo bot с backend на Django.
Позволяет управлять задачами через Telegram.

2. Архитектура

Проект состоит из следующих сервисов:

- Django backend — REST API, бизнес-логика
- PostgreSQL — основная БД
- Redis — брокер сообщений
- Celery worker — обработка фоновых задач
- Celery beat — планировщик уведомлений
- Telegram bot (Aiogram + Aiogram-Dialog)

3. Как работает уведомление

Celery Beat периодически проверяет задачи с истёкшим due_date.
Если задача не завершена и уведомление ещё не отправлено —
бот отправляет сообщение пользователю в Telegram.


4. Как запустить проект

git clone 
cd Django_tgbot
docker-compose up --build

5. Переменные окружения

Все переменные есть в .env-example. Можете создать у себя файл .env и вставить туда всё из .env-example
